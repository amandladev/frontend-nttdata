Technical Test

Este proyecto es un test técnico para demostrar el desarrollo de un sistema simple utilizando Angular para el frontend y Node.js + Express + Fastify + MongoDB para el backend.

Arquitectura Backend
Frontend Features

El frontend implementa 3 páginas principales:

Inicio de sesión

Permite a los usuarios registrados autenticarse utilizando su correo electrónico y contraseña. Después de un inicio de sesión exitoso, se almacena un token JWT y se utiliza para acceder a rutas protegidas.

Registro de usuarios

Permite a los usuarios registrarse proporcionando su nombre completo, correo electrónico, contraseña y número de teléfono. Se aplican reglas de validación para garantizar la calidad de los datos.

Lista de usuarios (Protegida)

Muestra una tabla con todos los usuarios registrados. Esta página está protegida, lo que significa que solo los usuarios autenticados pueden acceder a ella.

Tech Stack
  • Frontend: Angular 18+, Bootstrap 5.
  • Backend: Node.js, Express.js, Fastify, MongoDB and AWS.
  • Authentication: JWT.
